A shared repository of CTF tools and writeups maintained by .Hack's security team.

## What's Inside

- Reconnaissance scripts (port scanning, subdomain enumeration)
- Crypto challenge helpers (frequency analysis, XOR decoder)
- Web exploitation templates (SQLi, XSS payloads)
- Binary analysis cheat sheets
- Past competition writeups

## How to Use

Clone the repo and follow the README for setup instructions. Each tool has its own directory with usage examples.
